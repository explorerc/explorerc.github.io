git状态

git的状态分为三种：工作区->暂存区(索引区)->版本区

1.查看状态

当文件名是红色时—工作区
当文件名是绿色时—暂存区
不显示—-版本区

2.工作区->️暂存区

.表示将全部工作区的文件移入暂存区

3.暂存区->版本区

第一次提交的时候必须写-m和引号中的内容

4.暂存区->工作区
